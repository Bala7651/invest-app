     ```
     // After watchlist load completes:
     // Check if daily summary catch-up needed (fire-and-forget, not awaited)
     if (isCatchUpNeeded()) {
       const todayISO = getTodayISO();
       hasSummaryForDate(todayISO).then(has => {
         if (!has) {
           const { apiKey, modelName, baseUrl } = useSettingsStore.getState();
           if (apiKey) {
             useSummaryStore.getState().generateToday({ apiKey, modelName, baseUrl });
           }
         }
       });
     }
     ```
   - This is fire-and-forget — do NOT await the generateToday call. It runs in background while user interacts with watchlist.
   - The catch-up check runs after watchlist loads so that `useWatchlistStore.getState().items` has data for the generation.
