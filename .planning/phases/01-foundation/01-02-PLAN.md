       - `watchlist` table: id (PK autoIncrement), symbol (text, notNull, unique), name (text, notNull), sort_order (integer, notNull, default 0), created_at (integer timestamp, $defaultFn)
       - `daily_summaries` table: id (PK autoIncrement), symbol (text, notNull), date (text, notNull, ISO date), content (text, notNull, JSON blob), created_at (integer timestamp, $defaultFn)

    3. Create `drizzle.config.ts`:
       ```typescript

       export default defineConfig({

         out: './drizzle',
         dialect: 'sqlite',
         driver: 'expo',
       });
       ```

    4. Run `npx drizzle-kit generate` to produce initial migration SQL files in `./drizzle/`.
       Inspect the output: note the exact path of the generated migrations index file.

       ```typescript
       import { openDatabaseSync } from 'expo-sqlite';
       import * as schema from './schema';
       const expoDb = openDatabaseSync('invest.db', { enableChangeListener: true });
       export const db = drizzle(expoDb, { schema });
       ```
