  <action>
1. Create `CandleChart.tsx`:
   - Props: `{ data: OHLCVPoint[]; height: number; onCandleChange?: (candle: { open: number; high: number; low: number; close: number } | null) => void }`
   - Wrap wagmi `CandlestickChart.Provider` with data mapped to `{ timestamp, open, high, low, close }` (strip volume — wagmi doesn't use it).
   - Render `CandlestickChart` with given height.
   - Render `CandlestickChart.Candles` with `positiveColor="#00ff88"` (stock-up) and `negativeColor="#ff3366"` (stock-down).
   - Render `CandlestickChart.Crosshair` for press-and-hold interaction.
   - Use `CandlestickChart.useCandleData()` inside a nested child component to communicate the active candle back via `onCandleChange` callback. When crosshair activates, call `onCandleChange({open,high,low,close})`; when released, call `onCandleChange(null)`.
   - Bloomberg-style: no decorative gradients, clean appearance. Background should be transparent (parent provides bg-bg).
   - NOTE: If `CandlestickChart.useCandleData()` context issues arise (must be inside Provider), create an internal `CandleDataBridge` component nested inside Provider that reads the data and calls the callback.

2. Create `VolumeBar.tsx`:
   - Props: `{ data: OHLCVPoint[]; height: number; width: number }`
   - Use `@shopify/react-native-skia` `Canvas` and `Rect`.
   - Calculate `maxVolume` from data, then for each point draw a `Rect`:
     - x = `i * barWidth` (barWidth = width / data.length, with 1px gap)
     - height = `(point.volume / maxVolume) * height`
     - y = `height - barHeight` (bars grow upward)
     - color: `point.close >= point.open ? '#00ff88' : '#ff3366'` (green up, red down)
   - Handle empty data gracefully (render empty Canvas).

3. Create `TimeframeSelector.tsx`:
   - Props: `{ active: Timeframe; onSelect: (tf: Timeframe) => void; loading?: boolean }`
   - Render 5 pill-style `Pressable` buttons in a `flex-row` container.
   - Use Reanimated `useSharedValue` + `useAnimatedStyle` + `withTiming` for a sliding highlight (Animated.View absolutely positioned behind the active button).
   - Calculate pill width as container width / 5 (use `onLayout` to measure container width).
   - When tapped: immediately update `translateX.value` with `withTiming(index * pillWidth, { duration: 200 })`, then call `onSelect`. Instant visual feedback before data loads.
   - Active text: `text-bg font-bold`, inactive text: `text-muted`.
   - Highlight pill: `bg-primary rounded-full`.
   - Container: `bg-surface rounded-full p-1`.

4. Create `ChartSkeleton.tsx`:
   - Shimmer placeholder matching chart + volume area dimensions.
   - Use Reanimated `withRepeat(withTiming(0.3, { duration: 800 }), -1, true)` for opacity pulse.
   - Render a rounded rectangle (`bg-surface rounded-lg`) that pulses.
   - Accept `height` prop for flexible sizing.
  </action>
